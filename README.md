<h3>📖 Description</h3>

CineSage is a Retrieval-Augmented Generation (RAG) based movie recommendation system that combines semantic search with Google's Gemini model to generate intelligent and personalized movie recommendations.

<h3>✨ Features</h3>

- AI-powered movie recommendations
- Retrieval-Augmented Generation (RAG)
- Semantic Search using FAISS
- LangChain Retrieval Pipeline
- Gemini API Integration
- Modern React Frontend
- FastAPI Backend
- Rich movie metadata and AI-generated recommendations
